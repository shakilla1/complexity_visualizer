# Complexity Visualizer Homework

# Overview
This Flask app analyzes the execution time of algorithms and returns a JSON response with performance data and a base64-encoded graph.

# Folder Structure
- `app.py` : Flask application with `/analyze` endpoint
- `analysis/`:  Module containing algorithms, analyzer, and graph generator
- `requirements.txt`:  Python packages

# Usage
1. Install required libraries by using this command: "pip install -r requirements.txt"
